SVM pour la Classification de la Couverture Terrestre

Qu'est-ce que le SVM ?

La Machine à Vecteurs de Support (SVM) est un puissant algorithme d'apprentissage automatique utilisé pour classer différents types de terrains, tels que les forêts, les zones urbaines et les étendues d'eau.

Comment ça marche

  • Il trouve l'hyperplan optimal qui sépare les différentes classes.
  • Il maximise la marge entre les classes en utilisant des vecteurs de support.

Applications

  • Classification d'images satellitaires
  • Détection des changements de couverture terrestre
  • Planification et gestion urbaine

Avantages

  • Grande précision dans la classification
  • Efficace dans les espaces à haute dimension
  • Robuste contre le surapprentissage
